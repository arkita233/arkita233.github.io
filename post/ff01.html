条件概率、独立性与贝叶斯

条件概率

定义

在给定事件B发生的情况下，事件A发生的概率，记作：P(A|B)

单纯发生A,B的概率
A = M1 / N
B = M2 / N

B发生条件下A发生的概率
即事件A在B中所占比率
P(A|B) = M12 / M2

表达式

P(A|B) = P(AB) / P(B)

独立性

定义

P(A|B) > P(A)，说明B的发生促进了A的发生的可能

P(A|B) = P(A)，说明B的发生对A毫无影响

表达式 P(AB)=P(A)P(B) 推导

此时两事件独立
P(A|B) = P(AB) / P(B)
P(A|B) = P(A)

由两式子得出
P(AB)=P(A)P(B)

满足此式子说明两事件独立

全概率公式 - 由因到果

样本空间被切割为N个可数事件，并两两互斥

Bi x Bj=ϕ
B1+B2+B3…+Bn=Ω

根据图已知: P(A)=P(AΩ)
可推出: P(A)=P(AΩ)=P(AB1+AB2+AB3+…+ABn)

又有: P(A|B) = P(AB) / P(B)
可求出: P(A)=P(B1)P(A|B1)+P(B2)P(A|B2)+…+P(Bn)P(A|Bn)

意义 - 从小事件B推A总概率

已知各个小事件发生的情况下A发生的概率，求A总的发生概率

贝叶斯公式 - 由果推因

由全概率公式推导

P(Bi∣A)=P(ABi)P(A)=P(Bi)P(A∣Bi)P(A)P(B_i|A)=\frac{P(AB_i)}{P(A)}=\frac{P(B_i)P(A|B_i)}{P(A)}P(Bi​∣A)=P(A)P(ABi​)​=P(A)P(Bi​)P(A∣Bi​)​
=P(Bi)P(A∣Bi)P(B1)P(A∣B1)+P(B2)P(A∣B2)+...+P(Bn)P(A∣Bn)=\frac{P(B_i)P(A|B_i)}{P(B_1)P(A|B_1)+P(B_2)P(A|B_2)+...+P(B_n)P(A|B_n)}=P(B1​)P(A∣B1​)+P(B2​)P(A∣B2​)+...+P(Bn​)P(A∣Bn​)P(Bi​)P(A∣Bi​)​

意义 - 从A反推小事件B

已知各个小事件发生的情况下A发生的概率，反推A发生情况下某个小事件发生的概率
