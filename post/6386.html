    样本空间，随机事件

    两类现象

    确定性现象： 一定条件下必然发生
    随机现象： 一定条件，多种可能

    样本空间

    定义： 随机试验的所有结果构成的集合
    记为： S={e}
    元素e: 样本点

    随机事件

    样本空间 S 的子集 A 称谓随机事件 A

    EX: 公家车站等车人数
    事件A: 至少5人候车
    事件B: 候车人数不多于2人

    S={0,1,2…… }S=\left\{0,1,2\dots\dots\right\}S={0,1,2……}
    A={5,6,7…… }A=\left\{5,6,7\dots\dots\right\}A={5,6,7……}
    B={0,1,2}B=\left\{0,1,2\right\}B={0,1,2}

    1. 如果把S看做事件，则每次试验S总是发生，所以S为必然事件
    2. 如果一个事件只含有一个样本点，称其为基本事件
    3. 如果事件为空集**∅\varnothing∅， 则为不可能事件**

    事件的相互关系及运算

    事件的运算及关系

    1. A与B的和事件，A∪BA\cup BA∪B

      1. A∪B={x∣x∈A  或  x∈B}A\cup B=\{x\vert x\in A\;或\;x\in B\}A∪B={x∣x∈A或x∈B}
      2. A 与 B 至少有一个发生

    2. A与B的积事件，记为A∩B,A⋅B,ABA\cap B,A\cdot B,ABA∩B,A⋅B,AB

      1. A∩B={x∣x∈A  且  x∈B}A\cap B=\{x\vert x\in A\;且\;x\in B\}A∩B={x∣x∈A且x∈B}
      2. A 与 B 同时发生

    3. AB=∅ABAB=\varnothing ABAB=∅AB 互斥

    4. A与B的差事件A-B

      1. A−B  =  {x∣x∈A  且  x∉B}A-B\;=\;\{x\vert x\in A\;且\;x\not\in B\}A−B={x∣x∈A且x​∈B}

    5. A的逆事件，记为A‾\overline AA,也称A的对立事件

      1. A‾∪A=S\overline A\cup A=SA∪A=S
      2. A‾A=∅\overline AA=\varnothingAA=∅
      3. A‾‾=A\overline{\overline A}=AA=A

    6. 差事件的多种表示

      1. A  −  B  =  AB‾  =  A∪B−B  =  A  −  ABA\;-\;B\;=\;A\overline B\;=\;A\cup B-B\;=\;A\;-\;ABA−B=AB=A∪B−B=A−AB

    事件的运算定律

    1. 交换律
      1. A+B=B+AA + B = B + AA+B=B+A
      2. AB=BAAB = BAAB=BA
    2. 结合律
      1. A+(B+C)=(A+B)+C=A+B+CA + (B + C) = (A + B) + C = A + B +CA+(B+C)=(A+B)+C=A+B+C
      2. (AB)C=A(BC)=ABC(AB)C = A(BC) = ABC(AB)C=A(BC)=ABC
    3. 分配率
      1. A+BC=AB+ACA + BC = AB + ACA+BC=AB+AC$
      2. A(B+C)=AB+ACA(B + C) = AB + ACA(B+C)=AB+AC
    4. 摩根定律、对偶定律
      1. A∩B‾  =  A‾∪B‾\overline{A\cap B}\;=\;\overline A\cup\overline BA∩B=A∪B 并的非等于非的交 、 和的非等于非的积
        “A或B都不发生”等价于“非A和非B都发生”
      2. A∪B‾  =  A‾∩B‾\overline{A\cup B}\;=\;\overline A\cap\overline BA∪B=A∩B 交的非等于非的并 、 积的非等于非的和
        “A与B不同时发生”等价于“非A, 非B至少有一个发生”
    5. 其他
      等幂律
      1. A+A=AA + A = AA+A=A
      2. AA=AAA = AAA=A
      3. A+S=SA + S = SA+S=S
      4. AS=AAS = AAS=A
        0-1律
      5. A+∅=AA+\varnothing=AA+∅=A
      6. A∅=∅A\varnothing=\varnothingA∅=∅
        吸收律
      7. A⊂B则  A∪B=B  A∩B=AA\subset B则\;A\cup B=B\;A\cap B=AA⊂B则A∪B=BA∩B=A
        互补率
      8. A∩A‾=AA‾=∅A\cap\overline A=A\overline A=\varnothingA∩A=AA=∅
      9. A∪A‾=A+A‾=SA\cup\overline A=A+\overline A=SA∪A=A+A=S
        双重否定率
      10. A‾‾=A\overline{\overline A}=AA=A
    6. 拆互分解
      11. A∪B=A∪(B−A)A\cup B=A\cup(B-A)A∪B=A∪(B−A)
