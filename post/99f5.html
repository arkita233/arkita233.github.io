读取不定量数据

例子

# include <iostream>
int main()
{
	int sum = 0, value = 0;

	// 读取数据直到尾部
	while(std::cin >> value)
		sum += value

	std::out << "Sum is: " << sum << std::endl;
	return 0;
}

输入
3 4 5 6
输出
Sum is: 18
