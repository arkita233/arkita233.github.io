cpp primer 第二章 - 变量和基本类型
发表于 2020-07-21 14:53:20| 更新于 2020-07-21 20:22:44|c++
字数总计:1.3k|阅读时长: 5 分钟

基本内置类型

  1. 算术类型 arithmetic type
  2. 空类型 void

算术类型

  1. 整型 integral type - 包括字符和布尔
  2. 浮点型

bool 未定义
char 8位
wchar_t 16位
char16_t 16位 unicode字符
char32_t 32位
short 16位
int 16位
long 32位
long long 64位
float 6位有效数字
double 10位有效数字
long double 10位有效数字

带符号类型 和 不带符号类型 signed unsigned

char - signed char, unsigned char
由编译器决定，表示0-255，-127 - +127

类型转换

bool b = 42 // b 为 真
int i = b // i = 1
i = 3.14 // i = 3
double pi = i // pi = 3.0
unsigned char c = -1 // c 为 255 - 对256取模后的余数
signed char c2 = 256 // c2 未定义

不要混用带符号和不带符号的类型

字面值常量 literal

  1. 20 十进制
  2. 024 八进制
  3. 0x14 十六进制

字符和字符串

  1. ‘’ 字符字面值
  2. " " 字符串字面值

转义序列

\n 换行
\v 纵向制表符
\\ 反斜线
\r 回车
\t 横向制表符
\b 退格符
\? 问号
\f 进纸符
\a 报警符
\" 双引号
\' 单引号

泛化转义序列

\7 响铃
\12 换行
\40 空格
\0 空字符
\115 字符M
\x4d 字符M

指定字面值的类型 - 前缀和后缀

L'a'
u8"hi!"
42ULL
1E-3F
3.14159L

前缀

  1. u Unicode16 char16_t
  2. U Unicode32 char32_t
  3. L 宽字符 wchar_t
  4. u8 UTF-8 char

后缀

  1. u/U unsigned
  2. l/L long
  3. ll/LL long long
  4. f/F float
  5. l/L long double

变量

变量

定义

类型说明符 + 一个或多个变量名

初始值

初始化-创建变量时赋予初始值
赋值 - 擦除当前值，用新值代替

列表初始化 - {} - 编译时会有丢失值警告 Ex: long ld -> int a{dl}

int i = 0;
int i = {0};
int i{0};
int i(0);

默认初始化

定义变量时没有指定初始值，则被赋予默认值
未被初始化的内置类型变量是未定义的

变量声明和定义的关系

分离式编译共享代码 - 声明 declaration - extern 函数/变量 定义在其他文件

extern int i; //声明i而并非定义i
int j; //声明并定义
extern double pi = 3.14 // 定义

定义会分配存储空间，定义只能有一次，声明可以多次

标识符 identifier 和 变量命名规范

  1. 标识符体现实际含义
  2. 变量名用小写字母
  3. 自定义类用大写字母开头
  4. 标识符由多个单词时，应有明显区分（用下划线）

名字的作用域 scope - 一般由花括号分隔

全局作用域 - global scope
块作用域 - block scope

嵌套作用域

int r = 42;
int main()
{
	int u = 0;
	std::cout << r << u; // 42 0
	int r = 0
	std::cout << r << u; // 0 0
	std::cout << ::r << u; // 0 0

}

第三个使用了作用域操作符覆盖了默认的作用域规则

复合类型

引用和指针

引用

引用为对象起了另外一个名字
引用初始值必须是个同类型对象

int ival = 1024;
int &refVal = ival; // refVal指向ival
int &refVal2 // 报错，引用必须被初始化

指针 pointer

int *ip1, *ip2
double dp, *dp2;

获取对象的地址

指针存放的对象的地址提取，要用取地址符(&)

int ival = 42;	
int *p = &ival; // p存放ival的地址，或p是指向ival的指针

指针指向的类型要匹配

指针值

指针的值（地址）

  1. 指向一个对象
  2. 指向紧邻对象所占空间的下一个位置
  3. 空指针
  4. 无效指针

2,3,4会引发错误

利用指针访问对象

用解引用符(*)访问对象

int ival = 42;
int *p = &ival;
cout << *p;

空指针 null pointer

不指向任何对象

int *p1 = nullptr; // 初始化皆为0 最好使用这个
int *p2 = 0;
int *p3 = NULL;

赋值和指针

pi = &val // pi被改变，指向了val
*pi  = 0 // ival被改变

其他指针操作

任何非零指针对应的条件值都是true
==,!=比较类型相同指针的存放的地址是否相同

void 指针

void* 可以存放任意对象的地址，但不知道其中的类型

double obj = 3.14, *pd = &obj;
void *pv = &obj;
pv = pd;

指向指针的指针

int ival = 1024;
int *pi = &ival;
int **ppi = &pi;

const 限定符 - 不能被改变， const必须初始化

const int bufSize = 512;
bufSize = 512; // 赋值会引发错误

顶层 const

int *const p1 = &i; //p1 无法被改变
const int ci = 42; // ci 无法被改变， 顶层
const int *p2 = &ci; // 底层const, p2可以被改变

constexpr 和常量表达式

是不是常量表达式由数据类型和初始值决定

const int max = 20; // max 是
const int l = max + 1; // limit 是
int staff_size = 27; // staff_size 不是
const int sz = get_size(); // sz 不是

constexpr 变量 - c++11

constexpr int mf = 20; // 20是
constexpr int limit = mf + 1; // mf+1 是
constexpr int sz = size();// size 是 constexpr函数是才是

处理类型 - typedef

auto

auto item = val1 + val2 //根据val1,val2自动推断类型

decltype - 选择并返回操作数的数据类型

decltype(f()) sum = x ; // sum的类型为f的返回型

自定义数据结构

定义Sales_data类型

struct Sales_data {
	std::string bookNo;
	unsigned units_sold = 0;
	double revenue = 0.0;

}

头文件

预处理器，头文件保护

# ifndef SALES_DATA_H
# define SALES_DATA_H
# include <string>
struct Sales_data{
	std::string bookNo;
	unsigned units_sold = 0;
	double revenue = 0.0;
};
#endif
