基础

touch file 创建文件
cd 进入目录
pwd 查看当前所在目录
ls 查看目录
cat 打印文件

通用匹配符

例子1

touch asd.txt fgh.txt
ls *.txt

例子2

touch love_{1..10}_shiyanlou.txt

创建了是个文件

匹配符

  • 匹配 0 或多个字符
    ? 匹配任意一个字符
    [list] 匹配 list 中的任意单一字符
    [^list] 匹配 除 list 中的任意单一字符以外的字符
    [c1-c2] 匹配 c1-c2 中的任意单一字符 如：[0-9][a-z]
    {string1,string2,…} 匹配 string1 或 string2 (或更多)其一字符串
    {c1…c2} 匹配 c1-c2 中全部字符 如{1…10}

用户管理

查看自己

who am i
who mom likes
whoami

who 参数

-a 打印能打印的全部
-d 打印死掉的进程
-m 同am i，mom likes
-q 打印当前登录用户数及用户名
-u 打印当前登录用户登录信息
-r 打印运行等级

创建用户

su <user> 切换用户
sudo <cmd> 运行特权级别命令
su - <user> 切换用户同时切换环境变量和工作目录

sudo adduser <username> 添加用户
sudo passwd <username> 设置密码

退出用户

exit
Ctrl+D

用户组

查看用户组

groups <username>
cat /etc/group | sort
cat /etc/group | grep -E "<username>"

将其它用户加入 sudo 用户组

sudo usermod -G sudo <username>

删除用户

sudo deluser <> --remove-home

删除用户组

groupdel

文件权限

查看文件权限

长格式列出文件

ls -l

文件名意义

文件权限

cat <file name> 读文件
链接数
文件大小

ls 命令

ls -a 显示所有文件，包括隐藏文件

ls -al 

ls -dl 查看某个目录完整属性

ls -asSh 显示所有文件大小

变更文件所有者

sudo chown <username> <file name>

修改文件权限

chmod go-rw <file name>

g、o 还有 u 分别表示 group（用户组）、others（其他用户） 和 user（用户），+ 和 - 分别表示增加和去掉相应的权限。

adduser 和 useradd 的区别是什么

useradd 只创建用户，不会创建用户密码和工作目录，创建完了需要使用passwd <username>去设置新用户的密码。adduser 在创建用户的同时，会创建工作目录和密码（提示你设置），做这一系列的操作。

目录结构 文件操作

文件路径

cd .. 进入上级目录
cd ~ 进入home
pwd 获取当前路径

绝对路径

以根" / "目录为起点的完整路径，以你所要到的目录为终点，表现形式如： /usr/local/bin

相对路径

相对于你当前的目录的路径，相对路径是以当前目录 . 为起点，以你所要到的目录为终点，表现形式如： usr/local/bin

例子 以home为起点

# 绝对路径
$ cd /usr/local/bin
# 相对路径
$ cd ../../usr/local/bin

文件基本操作

新建空白文件

touch <file name>

新建目录

mkdir <dir name>

使用 -p 参数，同时创建父目录

mkdir -p father/son/grandson

mkdir 时 目录不能和存在文件同名

touch 文件与目录同名会更改文件夹的时间戳

复制

cp <file name> <destination>

cp 无法复制整个目录

递归复制

cp -r <dir name> <destination>

删除

删除文件

rm <filename>
rm -f <> 强制删除

删除目录

rm -r <dir>
rm -rf <>

移动与重命名

mv <file name> <dir>
mv <old file name> <new file name>

批量重命名

sudo apt-get install rename


使用通配符批量创建 5 个文件:
touch file{1..5}.txt

批量将这 5 个后缀为 .txt 的文本文件重命名为以 .c 为后缀的文件:
rename 's/\.txt/\.c/' *.txt

批量将这 5 个文件，文件名和后缀改为大写:
rename 'y/a-z/A-Z/' *.c

查看文件 cat,tac,nl

cat 正序

tac 倒序

cat -n <file>
显示行数

nl 添加行号并打印

  1. -b : 指定添加行号的方式，主要有两种：
    -b a:表示无论是否为空行，同样列出行号("cat -n"就是这种方式)
    -b t:只列出非空行的编号并列出（默认为这种方式）
  2. -n : 设置行号的样式，主要有三种：
    -n ln:在行号字段最左端显示
    -n rn:在行号字段最右边显示，且不加 0
    -n rz:在行号字段最右边显示，且加 0
  3. -w : 行号字段占用的位数(默认为 6 位)

使用 more 和 less 命令分页查看文件

more <file>

用于阅读

使用 head 和 tail 命令查看文件

tail /etc/passwd
tail -n 1 /etc/passwd
-n 接行数
-f 动态读取

file 查看文件类型

环境变量与文件查找

Shell变量

$ declare tmp
# 正确的赋值
$ tmp=shiyanlou
# 错误的赋值
$ tmp = shiyanlou
$ echo $tmp

环境变量

set 显示当前 Shell 所有变量，包括其内建环境变量（与 Shell 外观等相关），用户自定义变量及导出的环境变量。

env 显示与当前用户相关的环境变量，还可以让命令在指定环境中运行。

export 显示从 Shell 中导出成环境变量的变量，也能通过它将自定义变量导出为环境变量。

命令的查找路径与顺序
